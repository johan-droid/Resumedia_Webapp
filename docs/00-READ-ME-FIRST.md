## ğŸ†˜ **COMMON ISSUES & FIXES**

### **Issue: ZIP won't extract**
**Fix**: 
- Windows: Use 7-Zip (free) instead of built-in
- Mac: Double-click ZIP should work
- Linux: `unzip filename.zip`

### **Issue: npm install fails**
**Fix**:
```bash
# Try with legacy peer deps
npm install --legacy-peer-deps

# Or delete node_modules first
rm -rf node_modules package-lock.json
npm install
```

### **Issue: Port 3000 in use**
**Fix**:
```bash
# Use different port
npm run dev -- -p 3001
```

### **Issue: Can't find files**
**Fix**:
```bash
# Make sure you're in correct directory
cd resumedia-frontend
ls -la  # Should see src/ folder
```
